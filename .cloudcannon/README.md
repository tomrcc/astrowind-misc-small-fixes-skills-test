# Editing this site

Welcome! This site is built with AstroWind and edited in CloudCannon. This guide
orients you to what you can edit today.

## Blog

All articles live in [Blog](cloudcannon:collections/post).

- **Create** a post with the **+ Add** button — it opens in the content editor so
  you can start writing straight away.
- **Edit** an existing post visually, in the content editor, or via its data fields.
- Fill in the **title**, **excerpt** (a short summary used in listings and search
  results), **publish date**, **featured image**, **category**, and **tags**.
- Leave **draft** on while a post is unfinished — drafts don't appear on the live site.
- The **Advanced SEO** section is optional; ignore it unless you need to override how
  a specific post appears when shared on social media.

### Rich components in posts

Inside a post body you can insert these from the editor toolbar:

- **YouTube video** — paste a video ID
- **Vimeo video** — paste a video ID
- **Tweet** — paste a tweet URL
- **Logo** — the site logo

## Site settings

Site-wide settings live under [Site](cloudcannon:collections/config) → the
**config** file. From there you can change:

- The **site name** and default social-sharing details
- Blog behaviour (posts per page, related posts)
- The **colour theme** (system / light / dark)
